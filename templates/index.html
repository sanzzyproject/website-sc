Selamat datang di SannScraper API Interface. Alat ini dirancang untuk melakukan ekstraksi data web secara otomatis dan efisien.

1. Anti-Bot (User-Agent)

Fitur ini secara otomatis memutar User-Agent headers agar permintaan terlihat seperti berasal dari browser asli (Chrome/Firefox), menghindari pemblokiran oleh firewall sederhana.

2. Premium Proxy

Mengalihkan lalu lintas melalui jaringan proxy residensial. Ini menyembunyikan IP asli server dan memungkinkan akses ke situs yang dibatasi secara geografis.

3. Render JavaScript

Menggunakan browser headless untuk memuat situs web sepenuhnya, termasuk konten yang dirender oleh JavaScript (React, Vue, Angular) sebelum mengambil HTML-nya.

4. CSS Selector

Masukkan selector spesifik (contoh: div.product-price) untuk hanya mengambil data yang Anda butuhkan, bukan seluruh halaman HTML.
